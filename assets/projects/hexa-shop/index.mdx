## Project Overview

Hexa Shop is a full-stack e-commerce application built with modern web technologies. The goal was to create a fast, accessible, and visually appealing online shopping experience.

## Key Features

- **Product Catalog** – Browse products with filtering by category, price, and availability
- **Shopping Cart** – Add, remove, and update quantities with real-time price calculation
- **User Authentication** – Secure login/signup with session management
- **Responsive Design** – Optimized for mobile, tablet, and desktop
- **Search Functionality** – Fast product search with autocomplete suggestions

## Technical Implementation

### Frontend Architecture

The frontend is built with Next.js App Router for server-side rendering and optimal performance:

```tsx
// Product card component with optimized images
import Image from "next/image";
import { Product } from "@/types";

export function ProductCard({ product }: { product: Product }) {
  return (
    <div className="group relative overflow-hidden rounded-lg">
      <Image
        src={product.image}
        alt={product.name}
        width={400}
        height={400}
        className="object-cover transition-transform group-hover:scale-105"
      />
      <div className="p-4">
        <h3 className="font-semibold">{product.name}</h3>
        <p className="text-primary font-bold">${product.price}</p>
      </div>
    </div>
  );
}
```

### Database Schema

Using Prisma ORM with PostgreSQL for type-safe database operations:

```typescript
model Product {
  id          String   @id @default(cuid())
  name        String
  description String
  price       Float
  image       String
  category    Category @relation(fields: [categoryId], references: [id])
  categoryId  String
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt
}
```

### Cart State Management

Implemented a custom hook for cart management with localStorage persistence:

```typescript
export function useCart() {
  const [items, setItems] = useState<CartItem[]>([]);

  const addItem = (product: Product, quantity: number = 1) => {
    setItems((prev) => {
      const existing = prev.find((item) => item.id === product.id);
      if (existing) {
        return prev.map((item) =>
          item.id === product.id
            ? { ...item, quantity: item.quantity + quantity }
            : item
        );
      }
      return [...prev, { ...product, quantity }];
    });
  };

  return { items, addItem, removeItem, updateQuantity, total };
}
```
